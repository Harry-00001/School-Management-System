# School-Management-System
This is a school management system built using python and sqlite . It allow to store student info , update records and search or find student easily through a GUI

#Feature
1: New admission  ->   Add new student
Info need to enter :
Student name  , roll number , class , section ,  house (red , blue or othr) , post (headboy , prefect or none) , Bus (Bus no or own vehicle) , address , phone number , father name , mother name 

2 : Update student Infomation -> If any student info need to change
  Steps :
  -> Enter the admission number of the student 
  -> selct the field to update from the dropdown menu 
  ->Enter new value 

3: Find Student : -> To find student info or find admission number 
  This feature provude two option
  1: seach by admission number : Enter admission number and the system will disply alll info about that student in available 
  2: Find Admission number : If you don't know a student's admission number . Student can be search by using :
      * Name 
      * class 
      * Father Name 

4: Show all student -> This is is currently pending and may be add later
